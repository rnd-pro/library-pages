# Library Pages

Reusable documentation and GitHub Pages foundations for RND-PRO libraries.

The package will provide a semantic, progressively enhanced site shell, search,
base-path-safe navigation, and native JSDA build helpers. Product copy, diagrams,
animations, and demos remain in each library repository.

The public package contract is under active alpha development. JSDA Kit 1.6 is
consumed through its documented configuration and build interfaces and is not
modified by this project.
